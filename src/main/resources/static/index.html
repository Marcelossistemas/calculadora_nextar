Calculador_Nextar

Bem-vindo ao Calculador_Nextar!

Esta é uma aplicação Spring Boot que fornece uma API REST para avaliar expressões matemáticas.

Você pode utilizar os seguintes endpoints:

  • POST /api/evaliar: Avaliar uma expressão matemática.
  • GET /api/expressao: Listar todas as expressões avaliadas.

A documentação completa da API está disponível em /swagger-ui.html
